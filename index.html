      About

      Flash Cards is a small study tool I built while learning web development. It lets you create decks, add cards, and practice with a clean, simple interface — whether you're on desktop or mobile.

      I made this project as part of my journey into software engineering, and it taught me a lot about building real interfaces, handling state, and making things feel good to use. Hope you enjoy exploring it as much as I enjoyed putting it together.
